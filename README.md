# Program Pewarnaan Foto Hitam Putih Menggunakan OpenCV
Program ini digunakan untuk mengubah foto hitam putih menjadi foto berwarna menggunakan teknik:

Penyesuaian Intensitas (Brightness & Contrast)

Color Mapping

Penyesuaian Hue & Saturation (HSV)

Program ditulis menggunakan Python, OpenCV, dan Matplotlib.
# Fitur Utama
Membaca foto hitam putih (grayscale)

Meningkatkan intensitas (brightness & contrast)

Memberikan warna dasar menggunakan Color Map

Menyesuaikan Hue dan Saturation agar terlihat lebih natural

Menampilkan hasil sebelum & sesudah

Menyimpan foto hasil pewarnaan otomatis
# Struktur Program
Program terdiri dari 6 bagian utama:

Membaca gambar hitam putih

Penyesuaian intensitas (brightness & contrast)

Pemberian warna dasar menggunakan ColorMap

Penyesuaian warna (Hue & Saturation)

Menampilkan hasil

Menyimpan file hasil
# Hasil Akhir
Program menghasilkan foto hitam putih yang berubah menjadi:

Lebih cerah

Memiliki warna dasar

Saturasi & hue yang lebih natural
